### Basic Features
* Solar-System Manager
    * Add / Rename / Remove Solar-System
* Planet Manager
    * Add Planet / Moon
    * Modify Planet / Moon
    * Properties
        * Name
        * Distance (From Nearest Body)
        * Gravitional Pull
        * Color
* Mobile Component
    * Extension of Main App
    * Acts as a remote control
    * Changes are reflected on bigger screen
* Misc. Components
    * Email Verification
    * Forgot Password / Reset
    * Deleting an account
